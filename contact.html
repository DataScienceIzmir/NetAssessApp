Contact

Bug Reports/Feature Requests

The best way to report bugs or request new features is to open an issue on NetAssessApp's github page. This does require a github account. If you can't create a github account you can contact Eric Bailey to report an issue. Please include "NetAssessApp" in the subject of the email.
